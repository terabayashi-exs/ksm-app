# 実装済み機能一覧

このドキュメントでは、KSM-Appプロジェクトで実装されている全機能の詳細仕様を説明します。

## 📋 機能カテゴリ

### 💰 課金・プラン管理
- **[サブスクリプション・課金システム](./subscription-system.md)** - プラン管理、Square決済統合（Phase 1完了）

### 📊 順位・成績管理
- **[順位表システム](./standings-system.md)** - 事前計算、JSON形式保存、高速表示
- **[順位表統合システム](./standings-integration.md)** - 予選・決勝統合表示
- **[手動順位設定システム](./manual-rankings.md)** - 管理者による順位調整
- **[戦績表システム](./results-matrix.md)** - 対戦結果マトリックス表示

### 🎮 試合運営
- **[試合管理システム](./match-management.md)** - リアルタイム監視、QR認証、結果確定
- **[試合速報エリア](./live-news.md)** - 進行中試合のライブ表示
- **[スケジュールプレビュー](./schedule-preview.md)** - 自動生成、リアルタイム更新

### 🏆 大会管理
- **[大会ステータス管理](./tournament-status.md)** - 動的ステータス判定、日付ベース自動更新
- **[トーナメント進行システム](./tournament-progression.md)** - 自動進出、依存関係解決
- **[複数チーム参加機能](./multi-team.md)** - 同一マスターから複数エントリー

### 👥 チーム・選手管理
- **[CSV一括登録](./csv-import.md)** - マルチ行形式、バリデーション
- **[CSV一括登録（完全仕様）](./csv-import-complete.md)** - システム全体詳細
- **[辞退管理システム](./withdrawal-system.md)** - 基本仕様
- **[辞退管理システム（詳細）](./withdrawal-details.md)** - 影響度評価、承認・却下処理

### 🗄️ アーカイブ・データ管理
- **[大会アーカイブシステム](./archive-system.md)** - UI状態保存、バージョン管理、時点凍結

## 🎯 実装完了度

- 認証・権限管理
- 大会管理（CRUD、ステータス、公開設定）

- 試合管理・結果入力（リアルタイム、QR認証）

- アーカイブシステム（UI状態保存、バージョン管理）

- サブスクリプション課金機能（Phase 1完了、Phase 2以降未実装）
